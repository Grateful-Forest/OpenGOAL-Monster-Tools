# OpenGOAL-Monster-Tools
One-Click Monster Generator for Jak 1

![Navmesh](https://github.com/user-attachments/assets/21cf6645-7de2-492b-bf42-3eaaba11073d)


Supports:
- Easily draw areas where monsters chase Jak using Blender’s 3D interface.
-  Use any shape for navigation without worrying about triangulation.
- Click and draw paths to define routes monsters follow when idle.
- Precisely set obstacles in your space by drawing 3D spheres.
- Paint regions to specify gaps monsters jump over.
- Automatically format complex use-cases with a 1-click generation of all code.

Instructions:
1. Install the Blender Add-On
2. Enable Support for Navmesh [Here](https://github.com/LuminarLight/LL-OpenGOAL-ModBase/commit/4f897008fa2ec8809e04c2b32d5ef9c329afede8?diff=unified&w=0) (Thanks to LuminarLight for their Original Navmesh Implementation)
3. Select Navmesh in Blender and Generate
